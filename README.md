# 📊 Task 03 - Decision Tree Classifier

## 🎯 Objective
To build a machine learning model using Decision Tree to predict whether a customer will subscribe to a term deposit.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## 📌 Steps Performed
1. Loaded the dataset
2. Checked column names and structure
3. Converted categorical data into numerical format
4. Defined features (X) and target variable (y)
5. Split the dataset into training and testing sets
6. Built Decision Tree Classifier
7. Evaluated model using accuracy and confusion matrix
8. Visualized the decision tree

## 💡 Key Insights
- Decision Tree helps in understanding customer decision patterns  
- Model provides interpretable rules for prediction  
- Accuracy shows how well the model performs  
- Certain features strongly influence customer subscription
